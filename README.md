# 🤖 Smart Assistant – AI FAQ Chatbot

A web-based AI FAQ Chatbot that provides instant answers to user queries through keyword-based question matching and an interactive conversational interface.

## 🚀 Project Overview

This project focuses on improving user interaction by providing a chatbot that can quickly retrieve and display relevant information from a predefined knowledge base in a conversational format.

## ✨ Key Features

* Interactive chatbot interface
* Instant FAQ-based responses
* Keyword matching mechanism
* AI and technology-related knowledge base
* Real-time user interaction
* Responsive and modern UI
* Smooth chat experience
* Enter key support for sending messages

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome

## ⚙️ How It Works

1. Open the chatbot application.
2. Enter a question in the chat input field.
3. The chatbot analyzes the query using keyword matching.
4. The most relevant answer is retrieved from the FAQ database.
5. The response is displayed instantly in the chat window.
